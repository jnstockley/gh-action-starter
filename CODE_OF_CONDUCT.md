# Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment for everyone.

## Our Standards

- Be respectful and considerate in communication.
- Assume good intent and give constructive feedback.
- Avoid harassment, discrimination, or personal attacks.

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, and community discussions.

## Reporting

If you experience or witness unacceptable behavior, please report it via GitHub Security Advisories or by opening a private discussion with the maintainers.

## Enforcement

Maintainers may take any action they deem appropriate, including removing content or restricting participation.
